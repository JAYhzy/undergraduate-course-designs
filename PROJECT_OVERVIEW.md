# 项目说明

## 仓库定位

本仓库用于归档本科阶段课程设计、大作业和综合实践项目，偏作品集性质。

## 内容分类

- `coursedesign_I(c++)`：C++ 课程设计，包含考勤管理系统和课程设计资料。
- `coursedesign_II(java)`：Java 课程设计，包含图书馆/图书管理相关项目。
- `coursedesign_3(python)`：Python 课程设计，包含音乐偏好分析、数据可视化和电影推荐相关项目资料。
- `coursedesign_4`：Unity 3D 太空射击游戏课程设计。

## 项目方向

- C++ 控制台管理系统
- Java 面向对象课程设计
- Python 爬虫、Flask、数据清洗、ECharts 可视化
- 推荐系统入门项目
- Unity / C# 游戏开发

## 运行说明

不同课程设计运行环境不同：

- C++ 项目：使用 Visual Studio 打开 `.sln` 文件或使用对应编译器编译源码。
- Java 项目：使用 IntelliJ IDEA / Eclipse 导入源码或 Maven 项目。
- Python 项目：需要安装 Python 及项目依赖，部分项目还依赖 Node.js API 服务、MySQL/SQLite 或 CSV 数据文件。
- Unity 项目：使用 Unity 打开项目目录，`Library` 缓存会在首次打开时自动生成。

## 整理说明

仓库保留源码、项目配置、SQL、CSV、静态素材和必要运行资源。未上传课程提交报告、演示稿、IDE 缓存、编译产物、Unity `Library` 等不适合作为公开项目源码的内容。下载后如需运行，请根据具体项目补充安装环境和依赖。
