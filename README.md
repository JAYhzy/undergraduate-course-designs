# Undergraduate Course Designs

本科课程设计与综合大作业项目整理。

## 内容

- C++ 考勤管理系统课程设计
- Java 图书馆 / 管理类课程设计
- Python 音乐偏好分析与数据可视化项目
- Python 电影推荐相关项目资料
- Unity 3D 太空射击游戏课程设计

## 说明

本仓库用于归档本科阶段课程设计和综合实践项目。已过滤常见构建产物、缓存目录、IDE 临时文件、课程提交报告和演示稿，保留源码、项目配置、SQL/CSV 数据和必要运行资源。

`node_modules`、`target`、Unity `Library`、IDE 缓存和编译产物未纳入仓库；这些内容应在下载后通过依赖安装、编译或 Unity 重新生成。
